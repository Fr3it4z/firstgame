Tudo sobre o jogo do Pinto.

No que consiste o jogo?

Este jogo é para um trabalho de dois amigos, que querem brincar com bases de dados.

Neste jogo apenas é necessário matar todos os inimigos no menor tempo possivel.

Se fores o mais rápido vais estar no topo da tabela dos rankings.
